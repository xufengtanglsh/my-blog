python特有数据类型

特有数据类型
complex：复述类型
z1 = 3 + 4j
z2 = 1 + 2j
result = z1 * z2
print(result) # 输出: (-5+10j)

list：一般序列
    my_list = [1, 2, 3, "hello", 5.5]
tuple：	不可变序列
    my_tuple = (1, 2, 3, "hello", 5.5)
range： 数字序列
    range(5)      # 0, 1, 2, 3, 4
    range(1, 5)   # 1, 2, 3, 4
    range(1, 5, 2)  # 1, 3
dict： 映射
    person = dict(name="张三", age=25, city="北京")
    person.get("name") # 获取键值对
    person["name"] = "李四" # 修改键值对
    person.pop("age") # 删除键值对
    person.clear() # 清空字典   
set：集合不可重复
    set1 = {1, 2, 3, 4}
    set2 = {3, 4, 5, 6}
    print(set1 & set2)  # {3, 4} #交集
    print(set1 | set2)  # {1, 2, 3, 4, 5, 6} #并集
    print(set1 - set2)  # {1, 2} #差集
    
    numbers = [1, 2, 2, 3, 3, 4]
    unique_numbers = set(numbers)# 列表去重
    print(unique_numbers)  # {1, 2, 3, 4}
frozenset：不可变的集合
    frozen = frozenset([1, 2, 3]) # 创建冻结集合
    # frozen.add(4)  # 错误！不能修改

bytes：处理不可变二进制数据
    b = b'Hello'
    print(b[1:3])  # b'el'
bytearray：处理可变二进制数据
    ba = bytearray(b'Hello')
    ba[0] = 65 # 修改
    ba.append(33) # 添加   
    del ba[0] # 删除
memoryview：内存实图处理和共享二进制数据
    mv = memoryview(b'Hello')
    print(mv[0])# 访问 
    print(mv[1:3])# 切片

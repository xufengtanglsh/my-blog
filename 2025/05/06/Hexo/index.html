技術(Hexo)

HEROの常用コマンド

キャッシュクリア
hexo clean
リソース生成
hexo generate　→　hexo g
ロカール起動
hexo server　→　hexo s
デプロイ
hexo deploy
新たな文章作成
hexo new
セットコメント
hexo clean && hexo g -d

設定ファイル（_config.yml）も修正
①：
url: https://xufengtanglsh.github.io/my-blog/
permalink: :year/:month/:day/:title/
permalink_defaults:
pretty_urls:
trailing_index: true
trailing_html: true

②：
deploy:
type: git
repo: https://github.com/xufengtanglsh/my-blog.git
branch: master

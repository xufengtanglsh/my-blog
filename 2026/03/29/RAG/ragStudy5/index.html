langChain基础-LLM调用

①Gemini2.5 flash免费
https://aistudio.google.com/app/api-keys 注册APIKEY

②项目下创建.env文件
输入如下内容
GEMINI_API_KEY=XXXXXXXX

③python为了能够读到.env文件中设置APIKEY
pip install python-dotenv 安装

④调用LLM
pip install google-genai 安装

代码：
from google import genai
from dotenv import load_dotenv

load_dotenv()
genaiClient = genai.Client()
answer = generate_answer(question, reranked_docs, genaiClient)

def generate_answer(question: str, answer_docs: list, client: genai.Client)->str:
    prompt = f"""
    あなたはヘルプデスクのアシスタントです。以下の質問に対して、適切な回答を生成してください。
    質問に対して、以下のドキュメントを参考：
    {"\n".join(answer_docs)}

    質問: {question}
    """
    print(prompt)

    response = client.models.generate_content(
        model="gemini-2.5-flash",
        contents=prompt,
    )
    return response.text

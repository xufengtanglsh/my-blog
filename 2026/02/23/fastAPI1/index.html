技術(python)

fastAPI基础

检查python环境
python3 –version
创建虚拟环境
python3 -m venv venv
激活环境
source venv/bin/activate
激活环境中安装fastAPI
pip install fastapi uvicorn
激活环境中安装ORM
pip install sqlalchemy
工程下创建文件
main.py创建到venv虚拟环境目录以外

from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def root():
    return {"message": "Hello FastAPI"}

启动服务
uvicorn main:app –reload

访问
http://127.0.0.1:8000
http://127.0.0.1:8000/docs

停止服务
control + C

========router写法=========================
project/
├── main.py
├── routers/
│ ├── student.py
│ └── user.py

student.py
from fastapi import APIRouter
router = APIRouter()

@router.get("/students")
def get_students():
    return {"msg": "students"}

user.py
from fastapi import APIRouter
router = APIRouter()

@router.get("/users")
def get_users():
    return {"msg": "users"}

main.py
from fastapi import FastAPI
from routers import student
from routers import user

app = FastAPI()

app.include_router(student.router)
app.include_router(user.router)

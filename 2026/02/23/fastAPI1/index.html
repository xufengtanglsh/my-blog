fastAPI基础

检查python环境
python3 –version
创建虚拟环境
python3 -m venv venv
激活环境
source venv/bin/activate
激活环境中安装fastAPI
pip install fastapi uvicorn
激活环境中安装ORM
pip install sqlalchemy
工程下创建文件
main.py创建到venv虚拟环境目录以外

启动服务
uvicorn main:app –reload

访问
http://127.0.0.1:8000
http://127.0.0.1:8000/docs

停止服务
control + C

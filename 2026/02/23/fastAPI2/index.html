fastAPI参数

①路径参数
http://127.0.0.1:8000/students/100

@app.get("/students/{student_id}")
def get_student(student_id: int):
    return {"id": student_id}

特点：
必须写在 URL 路径中
自动类型转换
类型错误会自动报 422

②查询参数
http://127.0.0.1:8000/students?page=2&size=20

@app.get("/students")
def get_students(page: int = 1, size: Optional[int] = 10):
return {"page": page, "size": size}

特点：
可选参数可以给默认值
自动校验类型

③请求体参数（Body）
http://127.0.0.1:8000/students
{
“name”: “Tom”,
“age”: 20
}

from pydantic import BaseModel
class Student(BaseModel):
    name: str
    age: int

@app.post("/students")
def create_student(student: Student):
    return student

特点：
自动 JSON 解析
自动校验
自动生成文档
